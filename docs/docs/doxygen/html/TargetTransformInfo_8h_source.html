165 enum class TailFoldingStyle {
166  /// Don't use tail folding
167  None,
168  /// Use predicate only to mask operations on data in the loop.
169  /// When the VL is not known to be a power-of-2, this method requires a
170  /// runtime overflow check for the i + VL in the loop because it compares the
171  /// scalar induction variable against the tripcount rounded up by VL which may
172  /// overflow. When the VL is a power-of-2, both the increment and uprounded
173  /// tripcount will overflow to 0, which does not require a runtime check
174  /// since the loop is exited when the loop induction variable equals the
175  /// uprounded trip-count, which are both 0.

248  /// Underlying constants for 'cost' values in this interface.
249  ///
250  /// Many APIs in this interface return a cost. This enum defines the
251  /// fundamental values that should be used to interpret (and produce) those
252  /// costs. The costs are returned as an int rather than a member of this
253  /// enumeration because it is expected that the cost of one IR instruction
254  /// may have a multiplicative factor to it or otherwise won't fit directly
255  /// into the enum. Moreover, it is common to sum or average costs which works
256  /// better as simple integral values. Thus this enum only provides constants.
257  /// Also note that the returned costs are signed integers to make it natural
258  /// to add, subtract, and test with zero (a common boundary condition). It is
259  /// not expected that 2^32 is a realistic cost to be modeling at any point.
260  ///
261  /// Note that these costs should usually reflect the intersection of code-size
262  /// cost and execution cost. A free instruction is typically one that folds
263  /// into another instruction. For example, reg-to-reg moves can often be
264  /// skipped by renaming the registers in the CPU, but they still are encoded
265  /// and thus wouldn't be considered 'free' here.
266  enum TargetCostConstants {
267  TCC_Free = 0, ///< Expected to fold away in lowering.
268  TCC_Basic = 1, ///< The cost of a typical 'add' instruction.
269  TCC_Expensive = 4 ///< The cost of a 'div' instruction on x86.
270  };
