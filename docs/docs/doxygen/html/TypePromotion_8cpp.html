Detailed Description

This is an opcode based type promotion pass for small types that would otherwise be promoted during legalisation. This works around the limitations of selection dag for cyclic regions. The search begins from icmp instructions operands where a tree, consisting of non-wrapping or safe wrapping instructions, is built, checked and promoted if possible.
